# Web 端使用指南

> 在浏览器中直接使用 Nowen Note，无需安装桌面或移动客户端。

[官方网站](http://nowen.cn/) · [在线体验](http://note.nowen.cn/) · [教程中心](./README.md)

## 访问方式

在浏览器中打开你的 Nowen Note 服务地址：

- 局域网自部署：`http://<服务器或 NAS 的 IP>:3001`
- 反向代理：使用管理员配置的域名
- 在线体验：[http://note.nowen.cn/](http://note.nowen.cn/)

手机或其他电脑访问 NAS 时，不要填写 `localhost` 或 `127.0.0.1`，应使用 NAS 的局域网 IP、内网域名或公网地址。

> 官方网站和在线体验站当前使用 HTTP。自行公网部署时仍建议配置 HTTPS，以保护登录信息、Cookie 和附件传输。

## 推荐浏览器

| 浏览器 | 建议版本 | 支持情况 |
|---|---:|---|
| Chrome | 90+ | 完整支持 |
| Edge | 90+ | 完整支持 |
| Firefox | 90+ | 完整支持 |
| Safari | 15+ | 支持主要功能，部分下载和剪贴板行为受系统限制 |

出现编辑器、下载、剪贴板或文件选择问题时，优先使用最新稳定版 Chrome 或 Edge 复现。

## Web 端能力

Web 端覆盖主要功能：

- 富文本与 Markdown 双编辑器
- Markdown 实时预览、分屏和滚动同步
- 文档树、标签、收藏、搜索和回收站
- 双向链接、块引用、反向链接和知识图谱
- 图片、附件、文件管理和对象存储
- 待办、看板、日历、甘特图、提醒和模板
- 说说和思维导图
- AI 写作、Embedding 与 RAG 知识问答
- 工作区、分享、评论、公开知识空间和实时协作
- 版本历史、备份、导入导出和管理设置

管理员可以关闭 Web UI，仅保留 API 服务。遇到“网页端已被管理员关闭”提示时，需要改用桌面客户端连接，或由管理员重新开启网页访问。

## 常用快捷键

实际快捷键会根据编辑器模式和操作系统略有差异。

| 快捷键 | 常见作用 |
|---|---|
| `Ctrl/Cmd + K` | 打开命令面板或插入链接，取决于当前焦点 |
| `Ctrl/Cmd + N` | 新建笔记 |
| `Ctrl/Cmd + Shift + N` | 新建笔记本 |
| `Ctrl/Cmd + B` | 加粗 |
| `Ctrl/Cmd + I` | 斜体 |
| `Ctrl/Cmd + H` | 当前笔记搜索与替换 |
| `/` | 打开斜杠命令菜单 |

## 浏览器存储与离线缓存

Web 端会使用 IndexedDB 和本地存储保存登录状态、偏好和部分离线数据。

清理浏览器数据前请确认：

1. 编辑器显示已经保存。
2. 离线队列已经完成同步。
3. 重要内容已经在其他设备或服务端确认存在。
4. 已经创建可恢复的完整备份。

本地缓存用于提升速度和弱网体验，服务端数据库和附件目录仍是长期数据中心。

## 反向代理与公开链接

公网或域名部署时，管理员应配置正确的 `PUBLIC_WEB_ORIGIN`，否则分享链接、公开知识空间或回调地址可能使用内网 IP、错误端口或受保护地址。

还应按需配置：

- HTTPS
- `CORS_ORIGINS`
- 附件签名密钥
- 是否关闭旧版公开附件 URL
- 反向代理的 WebSocket 转发

详细说明见[完整部署指南](../deployment.md)和[安全设置](./security.md)。

## Web 与原生客户端怎么选

| 场景 | 推荐方式 |
|---|---|
| 临时使用、管理设置、任何设备快速访问 | Web |
| Windows / macOS / Linux 长时间写作 | Electron 桌面端 |
| 手机拍照上传、随时记录 | Android 客户端 |
| 调试网络和服务端问题 | 先用浏览器访问同一个地址 |

原生客户端连接失败时，先在同一设备的浏览器中打开服务器地址，可以快速判断是网络、服务端还是客户端配置问题。
