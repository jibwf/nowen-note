# 笔记导出图片

Nowen Note 的图片导出以**最终阅读预览**为输入，不会直接截图 Markdown 源码，也不会修改笔记正文、光标或编辑状态。

## 支持的格式

| 格式 | 适用场景 | 说明 |
| --- | --- | --- |
| PNG | 相册、截图、高清分享 | 默认推荐，无损输出 |
| JPEG | 长图、聊天软件分享 | 可调整质量以减小体积 |
| SVG | 设计工具、无损缩放 | 高级选项；Android 相册通常不接收 SVG |
| PDF | 超长文档、打印 | 图片导出窗口中可切换到 PDF |

Markdown 与富文本笔记使用同一套导出阅读样式。标题、列表、任务列表、表格、引用、分隔线、代码块、语法高亮、图片宽度与对齐都会从最终渲染结果生成。

## 超长内容

导出窗口提供三种策略：

- **自动**：普通内容生成一张长图；接近浏览器 Canvas 安全上限时自动分页。
- **长图**：在安全范围内生成一张图片；必要时自动降低倍率。
- **分页图片**：按临近的段落、标题、代码块、表格等内容边界拆分，避免在浏览器 Canvas 超限时产生空白图。

多张分页图片在网页或桌面端会打包为 ZIP；Android 保存到相册时会逐张写入。

## Android 保存位置

Android 原生应用提供三种明确入口：

1. **保存到相册**：写入 `Pictures/Nowen Note`，完成页展示实际文件名和位置。
2. **保存到系统文件**：打开 Android 文件选择器，由用户选择目录和文件名。
3. **分享到其他 App**：打开系统分享面板，可发送到微信、QQ、邮件或网盘等应用。

Android 10 及以上通过 MediaStore 分区存储写入相册，不申请全盘存储权限。保存完成后可以直接打开结果，也可以复制显示位置。

## 图片和字体加载

导出前会等待：

- 当前页面字体加载完成；
- 笔记图片完成下载与解码；
- 浏览器完成两帧布局计算。

本站附件会转换为自包含图片数据。外部图片如果因跨域、网络、鉴权或文件丢失无法读取，导出不会静默显示“成功”：

- 导出结果中会放置“图片加载失败”占位图；
- 完成页会显示失败数量、原始地址和具体原因；
- 其它可用内容仍可正常导出。

## 平台差异

- **Web / Electron**：使用浏览器下载目录；具体目录由浏览器或系统下载设置决定。
- **Android**：可选择相册、系统文件或分享，并返回 `content://` 文件地址供直接打开。
- **SVG**：适合保存到文件或分享，不提供“保存到相册”入口。
